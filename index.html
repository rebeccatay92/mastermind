Mastermind in Vanilla JavaScript

Instructions:

The goal of this game is to correctly guess the secret code made up of colored pegs. There are 6 possible colors, and the secret code can consist of repeated colors. To win, you need to get it right in 8 tries or less. Click on the pegs to select their color. Click the Check button to make a guess. The green number indicates the number of pegs with the correct color and in the correct position. The orange number indicates the number of pegs with the correct color.
